# Loop Engineering — 7-Day Interactive Course

Open `index.html` directly in a browser. No server, build step, or internet connection required.

## Contents

- Day 1 — Foundations: feedback systems, cybernetics, control theory
- Day 2 — Manual loops before automation
- Day 3 — Evaluation and rubrics
- Day 4 — One-shot prompting vs loop-engineered workflows
- Day 5 — Codifying a loop as a Hermes skill
- Day 6 — Bounded automation and human checkpoints
- Day 7 — Reflection, compounding, and loop portfolio design

## Features

- Sidebar navigation and local progress tracking
- Interactive loop visualization
- Idea-structuring workspace
- Rubric calculator
- Loop ROI estimator
- Hermes skill draft generator
- Automation plan generator
- Flashcards
- Quizzes with feedback
- Print/PDF stylesheet
